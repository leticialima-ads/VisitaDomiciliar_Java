# 🏥 Sistema de Acompanhamento Territorial em Saúde

## 📌 Sobre o Projeto

Este projeto é um sistema desktop desenvolvido em Java com foco no acompanhamento territorial em saúde pública, auxiliando profissionais no controle de pacientes e no registro de visitas domiciliares.

A aplicação permite identificar grupos prioritários para busca ativa, contribuindo para uma gestão mais eficiente e estratégica do cuidado em saúde.

---

## 🎯 Objetivo

Apoiar profissionais da saúde no monitoramento de pacientes em território, facilitando:

* Cadastro e organização de pacientes
* Registro de visitas domiciliares
* Identificação de pacientes prioritários
* Geração de relatórios para ações de saúde pública

---

## 🚀 Funcionalidades

### 👤 Gestão de Pacientes

* Cadastro de pacientes com dados pessoais e territoriais
* Listagem e exclusão de pacientes
* Busca por nome ou CPF

### 🏠 Visitas Domiciliares

* Registro completo de visitas
* Informações clínicas e sociais
* Identificação de pacientes acamados
* Registro de pressão arterial

### 🩺 Monitoramento de Saúde

* Controle de diabetes e hipertensão
* Acompanhamento de consultas
* Classificação de risco e prioridade

### 👩‍⚕️ Saúde por Perfil

* Saúde da mulher (preventivo, contraceptivos)
* Saúde da criança (caderneta de vacinação)
* Planejamento familiar (vasectomia)

### 📊 Relatórios Inteligentes

* Pacientes diabéticos
* Hipertensos
* Mulheres com preventivo atrasado
* Crianças com vacinação atrasada
* Pacientes acamados
* Grupos prioritários para busca ativa
